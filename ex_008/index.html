Principais Formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag < b > (forma: em desuso)

Nesta frase temos um termo em destaque, utilizando a tag < strong >. (semântica: HTML5).

Itálico/Ênfase

Nesta frase temos um termo em itálico, usando a tag < i >, não semântica. (em desuso - HTML4)

Nesta frase temos um termo em ênfase, utilizando a tag < em >. (semântica)

Texto Marcado

Podemos também criar um texto marcado utilizando a tag < mark >.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto Deletado e Texto Inserido

Podemos marcar um texto como excluído, para indicar que ele deve ser lido mas não deve ser considerado.

Podemos marcar um texto como inserido, para dar uma ênfase e indicar que ele foi adicionado depois.

Texto Sobrescrito e Texto Subscrito

Sobrescrito: Para inserir coisas do tipo X20 + 3.

Subscrito: Para inserir coisas do tipo H2O
